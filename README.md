# Forest-Cover-Classification
A machine learning classification model that predicts the type of forest cover in a given geographical area based on cartographic variables like elevation, soil type, slope, and wilderness area information. Can be used for environmental monitoring, forestry management, and land use planning. 
I have done Data cleaning, Preprocessing,Feature Engineering,EDA,Handled class Imbalace,Feature selection and model building Atlast but least used front end streamlit to predicted Forest Type.
